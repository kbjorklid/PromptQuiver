# Ink Components

A collection of core Ink components and external community components for use in Prompt Quiver.

## Core Components
- [<Text>](https://github.com/vadimdemedes/ink#text) - Displays text with support for colors, backgrounds, and styles (bold, italic, etc.).
- [<Box>](https://github.com/vadimdemedes/ink#box) - The essential layout component, providing Flexbox-based positioning and styling.
- [<Newline>](https://github.com/vadimdemedes/ink#newline) - Adds one or more newline characters within text components.
- [<Spacer>](https://github.com/vadimdemedes/ink#spacer) - A flexible space that expands to fill available space along the major axis.
- [<Static>](https://github.com/vadimdemedes/ink#static) - Permanently renders output above the main app, ideal for logs or completed tasks.
- [<Transform>](https://github.com/vadimdemedes/ink#transform) - Allows transformation of the string output of its children (e.g., for text effects).

## External Components
- [ink-text-input](https://github.com/vadimdemedes/ink-text-input) - Editable text input component.
- [ink-spinner](https://github.com/vadimdemedes/ink-spinner) - Spinner component for loading states.
- [ink-select-input](https://github.com/vadimdemedes/ink-select-input) - Select (menu) input component.
- [ink-link](https://github.com/vadimdemedes/ink-link) - Link component for terminal links.
- [ink-progress-bar](https://github.com/vadimdemedes/ink-progress-bar) - Progress bar component.
- [ink-multi-select](https://github.com/vadimdemedes/ink-multi-select) - Multi-select input component.
- [ink-confirm-input](https://github.com/vadimdemedes/ink-confirm-input) - Confirm (yes/no) input component.
- [ink-tab](https://github.com/vadimdemedes/ink-tab) - Tab component.
- [ink-gradient](https://github.com/vadimdemedes/ink-gradient) - Gradient text component.
- [ink-big-text](https://github.com/vadimdemedes/ink-big-text) - Component for rendering large text using fonts.
- [ink-image](https://github.com/vadimdemedes/ink-image) - Component for rendering images in the terminal.
- [ink-table](https://github.com/maticzav/ink-table) - Table component.
